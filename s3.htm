S3 Server Setup

What this install includes:

  • S3 Server Container on your local host
  • Auto configuration of a port in your machine
    • This allows you to communicate with your server in terms of storing data.

Purpose:

  • Allows you to store files on your server.

Modes of storing data in the server:

  • Drag and drop

Benefits of S3

  • Object storage
    • Manages data as objects vs. managing data as a file hierarchy and block storage.
    • Require less metadata than file systems when storing and accessing files.
    • Reduction of overhead managing of file metadata with object storage.
